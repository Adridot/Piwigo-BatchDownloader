$conf['batch_download_max_elements']
Maksimal værdi på Maksimalt antal fotografier pr. downloadsæt-skyderen.
Standardværdien er 1000.
$conf['batch_download_max_size']
Maksimal værdi på Maksimal størrelse på hvert arkiv-skyderen.
Standardværdien er 500.
$conf['batch_download_additional_ext']
Array indeholdende downloadbare filendelser (forskel på små og store bogstaver).
Standardværdien er $conf['picture_ext'].
$conf['batch_download_file_pattern']
Pattern for filenames in the archive. Available fields are '%id%', '%filename%', '%author%', '%dimensions%'.
Default value id %id%_%filename%_%dimensions%.
