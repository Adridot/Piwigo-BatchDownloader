$conf['batch_download_max_elements']
Maximalwert des Schiebereglers Maximale Anzahl an Bildern pro Satz. Standardwert 1000.
$conf['batch_download_max_size']
Maximalwert des Schiebereglers Maximale Größe des Archivs. Standardwert 500.
$conf['batch_download_force_pclzip']
Falls true, wird PclZip anstelle von ZipArchive als ZIP Bibliothek verwendet. Standardwert false.
$conf['batch_download_direct']
Falls true, wird die ZIP Datei per HTTP redirection anstelle PHP flow geliefert. Standardwert false.
$conf['batch_download_additional_ext']
Array der Dateiendungen zum Herunterladen (Groß/Kleinschreibung beachten). Standardwert $conf['picture_ext'].
