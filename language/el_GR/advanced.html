$conf['batch_download_max_elements']
Μέγιστη τιμή του Μέγιστος αριθμός φωτογραφιών ανά σετ λήψης slider. Η προεπιλογή είναι 1000.
$conf['batch_download_max_size']
Μέγιστη τιμή του Μέγιστο μέγεθος κάθε αρχείου slider. Η προεπιλογή είναι 500.
$conf['batch_download_force_pclzip']
Εαν ισχύει, εξαναγκάζει τη χρήση του PclZip αντι του ZipArchive ως βιβλιοθήκη ZIP. Η προεπιλογή είναι όχι.
$conf['batch_download_direct']
Εαν true, διαμοιράζει τα αρχεία ZIP με ανακατεύθυνση HTTP αντί της ροής PHP. Η προεπιλογή είναι όχι.
$conf['batch_download_additional_ext']
Πίνακα περιέχει επεκτάσεις αρχείων κατάληλων για κατέβασμα (case sensitive). Η προεπιλογή είναι $conf['picture_ext'].
