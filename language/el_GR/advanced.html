$conf['batch_download_max_elements']
Μέγιστη τιμή του Μέγιστος αριθμός φωτογραφιών ανά σετ λήψης slider.
Η προεπιλογή είναι 1000.
$conf['batch_download_max_size']
Μέγιστη τιμή του Μέγιστο μέγεθος κάθε αρχείου slider.
Η προεπιλογή είναι 500.
$conf['batch_download_additional_ext']
Πίνακα περιέχει επεκτάσεις αρχείων κατάληλων για κατέβασμα (case sensitive).
Η προεπιλογή είναι $conf['picture_ext'].
$conf['batch_download_file_pattern']
Pattern for filenames in the archive. Available fields are '%id%', '%filename%', '%author%', '%dimensions%'.
Default value id %id%_%filename%_%dimensions%.
