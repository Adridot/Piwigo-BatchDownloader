$conf['batch_download_max_elements']
Valore massimo del Numero massimo di foto per ogni download di set slider. Predefinito è 1000.
$conf['batch_download_max_size']
Valore massimo del Dimensione massima di ciascun archivo di slider. Predefinito è 500.
$conf['batch_download_force_pclzip']
Se true, impone l'utilizzo di PclZip anziché ZipArchive come libreria ZIP. Predefinito è false.
$conf['batch_download_direct']
Se true, fornisce i file ZIP con reindirizzamento HTTP invece di flusso PHP. Predefinito è false.
$conf['batch_download_additional_ext']
Array contenente le estensioni dei file scaricabili (differenzia maiuscole e minuscole). Predefinito è $conf['picture_ext'].
