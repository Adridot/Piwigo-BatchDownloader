$conf['batch_download_max_elements']
Максимальный уровень Максимальное количество изображений в наборе для закачки slider. По умолчанию 1000.
$conf['batch_download_max_size']
Максимальный уровень Максимальный размер каждого архива slider. По умолчанию 500.
$conf['batch_download_force_pclzip']
Если true, то применяется PclZip вместо ZipArchive в качестве Zip-архиватора. По умолчанию false.
$conf['batch_download_direct']
Если true, то разрешаются ZIP-файлы с перенаправлением HTTP, вместо PHP. По умолчанию false.
$conf['batch_download_additional_ext']
Массив-перечень расширений файлов, доступных для закачивания (с учетом регистра). По умолчанию $conf['picture_ext'].
